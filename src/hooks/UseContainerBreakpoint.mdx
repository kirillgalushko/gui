# useContainerBreakpoint

Возвращает текущий container breakpoint по фактической ширине элемента и набор boolean-флагов.

Хук использует `ResizeObserver`, поэтому подходит для сеток внутри узких контейнеров, сайдбаров, модалок и nested grid.

## Параметры

- `targetRef` — ref на HTML-элемент, ширину которого нужно отслеживать.

## Возвращаемое значение

Readonly объект:

```ts
{
  breakpoint: Breakpoint;
  isXS: boolean;
  isS: boolean;
  isM: boolean;
  isL: boolean;
  isXL: boolean;
  isXXL: boolean;
  isMobile: boolean;
  isGtXS: boolean;
  isGtS: boolean;
  isGtM: boolean;
  isGtL: boolean;
}
```

## Breakpoints

- `xs` — `0-575px`
- `s` — `576-767px`
- `m` — `768-1023px`
- `l` — `1024-1279px`
- `xl` — `1280-1599px`
- `xxl` — `1600px+`

## Пример

```ts
const targetRef = ref<HTMLElement | null>(null);
const container = useContainerBreakpoint(targetRef);

if (container.isGtM) {
  // ширина контейнера больше m
}
```
