# PasswordInput

Поле для ввода пароля на базе `Input`. В `rightAdornment` добавлена кнопка `Action`, которая переключает видимость пароля.

## Props

Поддерживает пропсы `Input`: `maxWidth`, `disabled`, `description`, `errorMessage`, `invalid`, `descriptionId`.

Дополнительные атрибуты прокидываются на внутренний input.

## Пример

```vue
<PasswordInput
  v-model="password"
  placeholder="Пароль"
  description="Минимум 8 символов"
/>
```
