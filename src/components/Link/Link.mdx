# Link

Компонент ссылки с типографикой из `Text`, состояниями и слотами для иконок.

## Props

- `typography?: Typography` — типографика из `Text`, по умолчанию `inherit`.
- `inline?: boolean` — включает inline-режим внутри текста.
- `underlined?: boolean` — показывает подчеркивание.
- `disabled?: boolean` — отключает ссылку, ставит `aria-disabled` и убирает из tab order.
- `enableVisited?: boolean` — включает visited-цвет.
- `mode?: 'inherit' | 'accent'` — цвет ссылки, по умолчанию `accent`.
- `Element?: 'a' | 'button' | 'span'` — тег компонента, по умолчанию `a`.

## Slots

- `default` — текст ссылки.
- `iconLeft` — иконка слева.
- `iconRight` — иконка справа.

## Пример

```vue
<Link href="/docs" underlined>
  <template #iconRight>
    <IconExternalLinkOutline />
  </template>
  Документация
</Link>
```
