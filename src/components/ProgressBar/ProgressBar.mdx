# ProgressBar

Индикатор прогресса с плавной анимацией заполнения. Может быть цельным или разделенным на сегменты.

## Props

- `progress?: number` — значение от `0` до `100`.
- `color?: string` — цвет заполнения, по умолчанию `accent`. Можно передать токен переменной (`positive`, `negative`, `danger`, `warning`), CSS-переменную (`--positive`) или любой CSS-цвет.
- `segments?: number` — количество сегментов, по умолчанию `1`.
- `height?: string` — высота, по умолчанию `8px`.
- `gap?: string` — расстояние между сегментами, по умолчанию `var(--gap-1)`.

## Пример

```vue
<ProgressBar :progress="75" :segments="4" color="positive" />
```
