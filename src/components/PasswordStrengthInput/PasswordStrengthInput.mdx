# PasswordStrengthInput

UI-компонент для создания сложного пароля. Внутри использует `PasswordInput` и `ProgressBar`.

Компонент не валидирует пароль сам. Правила передаются извне.

## Props

- `modelValue?: string` — значение поля.
- `rules?: PasswordStrengthRule[]` — список внешних правил.
- `progress?: number` — внешний прогресс от `0` до `rules.length`. Если не передан, считается по количеству выполненных правил.
- `progressSegments?: number` — количество сегментов прогресс-бара, по умолчанию `4`.
- `progressHeight?: string` — высота прогресс-бара.
- `progressGap?: string` — расстояние между сегментами.
- `weakColor?: string` — цвет слабого пароля, по умолчанию `negative`.
- `mediumColor?: string` — цвет среднего пароля, по умолчанию `warning`.
- `strongColor?: string` — цвет сильного пароля, по умолчанию `positive`.

Также поддерживает пропсы `PasswordInput`: `maxWidth`, `disabled`, `description`, `errorMessage`, `invalid`, `descriptionId`.

## PasswordStrengthRule

```ts
interface PasswordStrengthRule {
  id?: string | number;
  label: string;
  valid: boolean;
}
```

## Пример

```vue
<PasswordStrengthInput
  v-model="password"
  :rules="[
    { id: 'length', label: 'Минимум 8 символов', valid: password.length >= 8 },
    { id: 'number', label: 'Есть цифра', valid: /\d/.test(password) },
  ]"
/>
```
